# FlowerAI Roadmap 🗺️

## v1.0 — Foundation ✅ Released

- [x] 6-model ensemble (EfficientNet, ViT, ConvNeXt, Swin, CLIP, DINOv2)
- [x] MBIN research architecture
- [x] Disease detection (PlantVillage + PlantDoc)
- [x] Botanical RAG chatbot (LangChain + ChromaDB)
- [x] Growth forecasting (LSTM + TFT)
- [x] Smart garden assistant
- [x] Video analytics (YOLOv9 + DeepSORT)
- [x] XAI dashboard (Grad-CAM++, SHAP)
- [x] FastAPI backend with JWT/RBAC/rate limiting
- [x] Next.js frontend
- [x] Flutter mobile app
- [x] Docker + Kubernetes + Terraform
- [x] MLflow + W&B + Airflow MLOps pipeline
- [x] Prometheus + Grafana monitoring

## v1.1 — Enhanced Intelligence (Q3 2024)

- [ ] **Hyperspectral Imaging** — NIR/UV band support for invisible disease markers
- [ ] **Federated Learning** — Privacy-preserving training across distributed garden devices
- [ ] **Real-time Video Streaming** — WebRTC-based live camera classification
- [ ] **MBIN v2** — Extended architecture with temporal video encoder
- [ ] **Fine-grained Pest Detection** — 50+ insect pest classes (aphids, spider mites, thrips)
- [ ] **Soil Analysis Integration** — API connectors for IoT soil sensors (EC, N/P/K)
- [ ] **AR Identification** — WebXR overlay for in-garden plant identification
